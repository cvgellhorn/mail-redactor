mail-redactor
===========
> Fool all the evil mail harvesting bots by randomly splitting and wrapping mail addresses

# Installation
    npm install mail-redactor

# Usage
``` js
const mailRedactor = require('mail-redactor');
console.log(mailRedactor('contact@website.com'));
```
